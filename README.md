dtools
======

A set of useful tools.

## Installation
	
	npm install dookie-tools --save

## Usage
	
	var dtools = require('dookie-tools');

	dtools.randomInt(1,14);
	dtools.completeZerosLeft(2,4);
	dtools.numberArray(2013,3,true);
